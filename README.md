## Proyectos destacados

### 🔹 Sistema de Gestión de Titulación Académica (SGT)

Sistema web diseñado para digitalizar y optimizar el proceso de titulación en instituciones educativas.

- Gestión integral de expedientes
- Seguimiento de procesos y control de estatus
- Generación de reportes y documentación
- Arquitectura cliente-servidor con API REST
- Enfoque en escalabilidad y mantenimiento institucional

🔗 https://github.com/PachecovBryan/titulacion-management-system

### 🔹 IoT Intelligent Safety System

Sistema IoT basado en ESP32 para monitoreo de condiciones ambientales y detección de riesgos.

- Detección de gas, fuego y temperatura
- Integración con Telegram para alertas en tiempo real
- Activación automática de actuadores
- Arquitectura modular (sensores, controladores, servicios)

🔗 https://github.com/PachecovBryan/iot-intelligent-safety-system
